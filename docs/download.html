Installation Guide


Windows Installation
1 Download the .7z archive from the links above
2 Extract using 7-Zip or Windows built-in extractor
3 Navigate to the extracted folder
4 Run GK Install Builder.exe



Linux Installation
1 Download the .7z archive from the links above
2
Install p7zip if needed:
sudo apt install p7zip-full    # Debian/Ubuntu
sudo dnf install p7zip         # Fedora/RHEL
sudo zypper install p7zip      # SUSE
3
Extract the archive:
7z x gk-install-builder-*.7z
4
Make executable and run:
cd gk-install-builder-*
chmod +x "GK Install Builder"
./"GK Install Builder"
